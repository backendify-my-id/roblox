# Stack Wizard Skill

Skill ini bertugas membimbing pembuatan boilerplate proyek baru dengan efisien.

## Aturan Utama:
1. Selalu gunakan `npx -y` untuk instalasi framework non-interaktif.
2. Inisialisasi struktur file modular dari awal (misal: folder `/src`, folder `/components`, dll).
3. Selalu periksa opsi bantuan cli menggunakan `--help` sebelum menginisialisasi script baru.
