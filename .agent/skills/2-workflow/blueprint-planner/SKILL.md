# Blueprint Planner Skill

Skill ini mengatur tata cara pembuatan rencana implementasi kerja agar selalu presisi dan terstruktur.

## Aturan Utama:
1. Selalu buat draf rencana kerja menggunakan cetakan standar `TEMPLATE_PLAN.md`.
2. Kelompokkan rencana aksi menjadi: Prasyarat Database, Modifikasi Backend/API, Modifikasi Frontend/UI, dan Kasus Uji (Test Cases).
3. Ajukan rencana kerja sebagai draf Artifact yang memerlukan verifikasi manual dari pengguna sebelum mulai menulis kode.
