# Doc Generator Skill

Skill ini bertugas memastikan dokumentasi teknis proyek selalu rapi, akurat, dan mudah dipahami.

## Aturan Utama:
1. Selalu gunakan format markdown standar GitHub.
2. Sediakan contoh request dan response HTTP yang konkret pada spesifikasi API.
3. Gunakan diagram alur (Mermaid diagram) jika menjelaskan alur arsitektur data plane yang rumit.
